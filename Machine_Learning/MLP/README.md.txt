# This folder contains the Multi-layer Perceptron design and code for our project.

## The **Final_version_used** folder contains the scaler and mlp pickle files that were actually used during our final demo.

These pickle files are loaded at external_comms/fpga_server.py to do ML classification on FPGA server

## The **intermediate_versions** folder contains the intermediate pickle files and design that we had used. Saved here as reference

## The two folders **window_size_50_ML_code** and **window_size_100_ML_code** contain the code for extract data and training/testing the ML models
